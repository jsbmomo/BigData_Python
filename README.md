# 공공데이터 포털 데이터 분석


학교 빅데이터 수업 과제로 제출한 Python 빅데이터 분석.
주제는 자유였으므로 공공데이터 포털에서 아래와 같은 데이터를 가져왔습니다.

  - 울산 연도별 교통단속 현황
  - 울산 무인단속장비 설치 현황
  
울산 연도별 교통단속 현황은 울산의 각 연도별 교통단속 현황을 pie 그래프로 출력하거나
울산의 14 ~ 18년도 교통단속 현황을 bar 그래프로 출력하기 위해 사용하였습니다.


울산 무인단속장비 설치 현황은 본래 무인단속장비가 설치된 위도와 경도 데이터를 가져오고자
했으나, 무인단속장비가 위치한 지역명만 있기 때문에, 위도와 경도 데이터는 구글 지도에 지역명을
입력하여 구한 후, CSV 파일에 직접 입력하였습니다(단, 특정 지역 하나만).
